Fais-moi un ToC d'un cours magistral sur le machine learning.
Il faut que le cours puisse permettre à n'importe quel bachelor scientifique de devenir expert dans le domaine du machine learning.
Pour cela, il doit avoir des chapitres dédiés qui explique les notions mathématiques qui dépasse le niveau collège-lycée.
Aussi, il faut que des exercices soient présentés pour tester les notions apprises et les mettre en pratique afin d'améliorer la mémorisation.


Pour cela, il faut aller du plus global / général, vers le plus précis / spécifique, et cela pour chaques chapitres / grandes parties.
Je perçois un axe possible pour ce Toc, sur lequel tu peux te baser, mais qu'il faudrait approfondir :
- Histoire du ML
- Paramétrique : Modèle linéaire / Modèles bayésien / Réduction de dimension / SVM / Auto-regression
- Non Paramétrique : Arbre décisionnel / Ensembliste / Voisin proche (nearest neighbor) / Clustering
- Réseau de neurone : ANN / CNN / RNN / GAN / Diffusion / Transformer / Auto-encodeur
- Reinforcement ML : Value Iteration / Policy Iteration / Genetic Algorithm
